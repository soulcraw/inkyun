# In-Kyun Kim homepage

This is a Quarto website for In-Kyun Kim.

## Preview locally

```bash
quarto preview
```

## Render

```bash
quarto render
```

The rendered website will be created in the `docs/` folder.

## GitHub Pages

Use a GitHub repository named:

```text
soulcraw.github.io
```

Then set GitHub Pages to publish from:

```text
main branch / docs folder
```

## Pages

- `index.qmd`: homepage
- `cv.qmd`: CV page
- `research.qmd`: publications, preprints, and books
